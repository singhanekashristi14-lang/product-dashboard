# Product Portfolio Intelligence Dashboard

## Objective
Analyze 600 SKUs to identify:
- Loss-making products (Kill)
- Optimization opportunities (Improve)
- High-potential products (Invest)

## Approach
- Cleaned data and ensured consistency
- Calculated margins, profits, and performance metrics
- Created decision logic based on:
  - Profitability
  - Sales volume
  - Customer ratings

## Key Insights
- ~22% products are loss-making
- ~68% need optimization
- ~10% are high-potential

## Tools Used
- Google Sheets
- Data Analysis
- Dashboard Design

## Output
- Interactive dashboard
- Decision segmentation
- Top performing & loss-making SKUs
